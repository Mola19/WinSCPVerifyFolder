This is an WinSCP extension to compare a local folder with a remote one.
It generates hashes for each file in the folders (and subfolders).

For information on how to install it: https://winscp.net/eng/docs/extension

It is losely based on the preinstalled VerifyFileChecksum and isn't particularly fleshed out right now.

SSH keys won't work with extensions by default, you'll need to use Pageant (which comes with WinSCP)
https://winscp.net/eng/docs/ui_pageant
